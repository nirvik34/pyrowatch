## 🏗️ Technical Architecture

### 📊 Data Pipeline (The "Insight")
- **NASA FIRMS**: Fetches Near-Real-Time (NRT) thermal anomalies via VIIRS/MODIS satellites.
- **NOAA CDO**: Atmospheric stream integration (Wind speed, direction, humidity, temperature).
- **Featherless AI**: Vision-language processing and automated Situation Report generation (Mistral-7B).

### 🧠 ML Model
- **Type**: Multivariate LSTM (Long Short-Term Memory).
- **Features**: Normalized FRP (Fire Radiative Power), Wind Vectors, Thermal Gradients, and Humidity Spikes.
- **Performance**: High-fidelity fire line prediction capable of flagging high-risk zones 6 hours before detection by traditional means.

## 🚀 Deployment & Local Setup

### 1. Environment Configuration
Create a `.env` in the root directory:
```bash
NASA_FIRMS_API_KEY=your_key
NOAA_CDO_TOKEN=your_token
FEATHERLESS_API_KEY=your_key
```

### 2. Backend Setup
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn api.main:app --reload --port 8000
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
Accessible at: `http://localhost:5173`
